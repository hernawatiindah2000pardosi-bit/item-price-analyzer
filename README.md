# Item Price Analyzer (C++)

A terminal-based C++ application designed to process, analyze, and manage retail item prices. This project demonstrates foundational knowledge of array manipulation, basic arithmetic calculations, and manual conditional searching algorithms without relying on built-in optimization libraries.

## 🌟 Key Features
- **Dynamic Data Collection:** Allows users to input a custom number of retail items dynamically.
- **Automated Price Aggregation:** Instantly calculates the overall accumulated price and evaluates the mathematical average.
- **Extreme Value Analysis:** Programmatically detects the maximum and minimum price boundaries alongside their exact index locations within the dataset.

## 🛠️ Programming Concepts Implemented
- **Control Flow & Structured Logic:** Clean utilization of conditional structures (`if-else`) and programmatic loops (`for-loops`).
- **Data Collections:** Practical use of array data structures for linear item price mapping.
- **Problem Solving:** Manual algorithm development for statistical search optimization (Max/Min indexing tracking).
